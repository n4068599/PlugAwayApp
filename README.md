PlugAway - Plugin Marketplace


---

**Objective**


Plug Away is plugin market place that will allow users to download plugin for their appropirate platform. On the backend it uses Node.js and has connection to monggodb to store users and plugin details. On the front end it used React.JS

User on first login will be able to register. Once registered one can login and add, retrieve , update or delete their plugins.

Test script for CI/DI is also included
CI - ./backend/test/teset.js
DI - ./github/workflows/ci.yml
